# Timee — Crystals for the Present Moment 🌙🐝

A premium independent brand selling mindful crystals to the global market.
Built as a static site, ready to deploy on GitHub Pages with your custom domain.

## Quick Start

1. Push this repo to GitHub
2. Go to Settings → Pages → Deploy from `main` / `/docs`
3. Add your custom domain: `timee.world`
4. Point your domain's DNS to GitHub Pages IPs

## Tech Stack

- Pure HTML + CSS + JS (no framework)
- Google Fonts: Playfair Display + Inter
- Hosted on GitHub Pages (free)
- Ready for Stripe/Gumroad/Lemon Squeezy integration
